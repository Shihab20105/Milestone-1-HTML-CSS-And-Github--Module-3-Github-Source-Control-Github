Introduction to Git init, git add, git commit

To Create or Start a module:

  1. Create a Repository
  2. Create a new folder
  3. Then open the folder in VS Code
  4. Create a new test file
  5. Open terminal in VS Code
  6. Repository--VS Code(Folder)

Then-

git init

git add

git commit -m"Learned about how to Create or Start a Module"

git push
